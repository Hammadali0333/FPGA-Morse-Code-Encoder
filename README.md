# FPGA Morse Code Encoder

## Overview

This project implements a real-time Morse Code Encoder on the Digilent Nexys 4 DDR FPGA board using Verilog HDL. The system converts decimal digits into Morse code and outputs the encoded sequence using LEDs with accurate timing generated by a finite state machine (FSM).

The design supports two operating modes:

- **Single Digit Mode** – Encodes one decimal digit (0–9).
- **Number Mode** – Encodes up to six decimal digits stored inside a FIFO buffer.

The project demonstrates FPGA-based digital design, finite state machine implementation, hardware timing control, simulation, synthesis, and real-time hardware verification.

# System Architecture

The design consists of the following modules:

- Morse Encoder
- Top Module
- FIFO Storage
- Debouncer
- Clock Divider

---

# Design Flow

1. User enters a digit using switches.
2. FSM reads the input.
3. Digit is converted into Morse Code.
4. Timing controller generates dot and dash durations.
5. LEDs display the Morse code sequence.
